# Pop Corn website with [React][react-docs] + [Redux][react-redux-docs] + [Tailwind CSS][tailwind-css-docs]

## Prerequisites

* Node.js v18 or later - [installation][nodejs-install]

## Getting Started

To get started with the project, install dependencies:

```sh
npm install 
```

## Launch

```sh
npm run start
```

## Build

```sh
npm run build
```
